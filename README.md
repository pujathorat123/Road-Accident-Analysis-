# Road Accident Analysis Project

## Tools Used
- Microsoft Excel
- Pivot Tables
- Charts
- Dashboard

## Project Overview
This project is based on analyzing road accident data using Microsoft Excel.  
The main purpose of this project is to convert raw accident data into meaningful insights with the help of charts, pivot tables, and an interactive dashboard.
This analysis helps in understanding accident patterns, casualty trends, vehicle involvement, road type conditions, and yearly comparisons.


## Key Insights
- Total accidents by year
- Casualties by vehicle type
- Urban vs Rural accidents
- Day vs Night accidents
